# Ghota

> Plataforma de comunidades privadas com chat em tempo real, gestão de membros, cargos e planos de subscrição pagos via Stripe.

> ⚠️ **Lê [AGENTS.md](../../../AGENTS.md) primeiro** — contém instruções obrigatórias sobre como usar esta vault como memória persistente.

**Stack:** Laravel 13 + React 19 + Inertia.js + Vite 6 + Tailwind CSS 4 + PostgreSQL 17 + Laravel Reverb + Stripe + Docker.
